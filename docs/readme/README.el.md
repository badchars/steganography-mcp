## Το Πρόβλημα

Η στεγανογραφία είναι η τέχνη της απόκρυψης δεδομένων μέσα σε φαινομενικά αθώα μέσα &mdash; εικόνες, αρχεία ήχου, έγγραφα, ακόμη και κείμενο Unicode. Χρησιμοποιείται σε διαγωνισμούς CTF, έρευνες ψηφιακής εγκληματολογίας, κρυφά κανάλια επικοινωνίας και κακόβουλα φορτία. Η ανίχνευσή της απαιτεί συνδυασμό στατιστικής ανάλυσης, αποκωδικοποίησης μορφών αρχείων, μέτρησης εντροπίας και εξειδικευμένης γνώσης.

```
Παραδοσιακή ροή εργασίας στεγανογραφικής ανάλυσης:
  ανίχνευση στεγανογραφίας εικόνας  ->  zsteg + stegsolve (2 εργαλεία, Ruby + Java)
  ανάλυση chi-square                 ->  προσαρμοσμένο Python script
  ανάλυση RS                         ->  προσαρμοσμένος κώδικας MATLAB/Python
  εγκληματολογία JPEG DCT            ->  stegdetect (εγκαταλελειμμένο C εργαλείο, 2004)
  εξαγωγή δεδομένων LSB              ->  zsteg + steghide + openstego (3 εργαλεία)
  στεγανογραφία ήχου                 ->  Audacity χειροκίνητα + scripts
  ανίχνευση μηδενικού πλάτους        ->  web εργαλεία + χειροκίνητος έλεγχος
  εγκληματολογία αρχείων / binwalk   ->  binwalk + foremost + xxd (3 εργαλεία)
  μεταδεδομένα EXIF                  ->  exiftool (εξάρτηση Perl)
  ανίχνευση κωδικοποίησης            ->  CyberChef web UI + χειροκίνητη εικασία
  ─────────────────────────────────
  Σύνολο: 10+ εργαλεία, 5+ γλώσσες, ώρες χειροκίνητης συσχέτισης
```

Το **steganography-mcp** δίνει στον AI agent σας 60 εργαλεία σε 7 κατηγορίες μέσω του [Model Context Protocol](https://modelcontextprotocol.io). Ο agent εκτελεί στεγανάλυση εικόνας, εγκληματολογία JPEG, ανάλυση ήχου, ανίχνευση στεγανογραφίας κειμένου, εγκληματολογία αρχείων, ανάλυση εγγράφων και αναγνώριση κωδικοποίησης &mdash; όλα σε μία μόνο συνομιλία, εκτελούμενα 100% τοπικά χωρίς καμία εξάρτηση από εξωτερικές υπηρεσίες.

```
Με το steganography-mcp:
  Εσείς: "Ανάλυσε αυτή την εικόνα CTF πρόκλησης για κρυμμένα δεδομένα"

  Agent: -> img_detect: Chi-square p=0.0001 (εντοπίστηκε LSB ενσωμάτωση),
            η ανάλυση RS εκτιμά ρυθμό ενσωμάτωσης 42%, ανωμαλία εντροπίας
            στο κάτω-δεξιά τεταρτημόριο
         -> img_lsb_extract: Εξήχθησαν 847 bytes από RGB LSBs
         -> crypto_detect: Τα εξαχθέντα δεδομένα είναι κωδικοποιημένα σε Base64
         -> crypto_decode: Αποκωδικοποίηση σε "FLAG{hidden_in_plain_sight_2024}"
         -> img_known_tools: Ταίριασμα υπογραφής OpenStego

         "Η εικόνα περιέχει στεγανογραφία LSB ενσωματωμένη με OpenStego.
          Η δοκιμή chi-square επιβεβαιώνει αντικατάσταση LSB και στα τρία
          κανάλια RGB με ρυθμό ενσωμάτωσης 42%. Το κρυμμένο φορτίο είναι
          κωδικοποιημένο σε Base64 και αποκωδικοποιείται στη σημαία:
          FLAG{hidden_in_plain_sight_2024}"
```

## Τι Μπορεί να Κάνει η AI

### Επίλυση Προκλήσεων CTF

```
Εσείς: "Λύσε αυτή την πρόκληση στεγανογραφίας CTF — η εικόνα βρίσκεται στο challenge.png"

Agent: -> img_detect: Η στεγανάλυση chi-square ανιχνεύει LSB ενσωμάτωση
          (p=0.00012), η ανάλυση RS εκτιμά ρυθμό ενσωμάτωσης 38%,
          ανωμαλία εντροπίας στα blocks (12,4)-(15,8)
       -> img_lsb_extract: Εξήχθησαν 1.247 bytes από τα LSBs καναλιών RGB
       -> crypto_detect: Τα δεδομένα ταιριάζουν με μοτίβο Base64 (εμπιστοσύνη: 0.95)
       -> crypto_decode: "Q1RGe3N0M2cwX21hc3Rlcn0=" -> "CTF{st3g0_master}"
       -> img_known_tools: Ανιχνεύτηκε υπογραφή OpenStego σε PNG chunks

       "Βρέθηκε η σημαία: CTF{st3g0_master}
        Η εικόνα χρησιμοποιεί στεγανογραφία LSB ενσωματωμένη από OpenStego.
        Το φορτίο ήταν κωδικοποιημένο σε Base64 στα λιγότερο σημαντικά
        bits RGB με ρυθμό ενσωμάτωσης 38%."
```

### Έρευνα Ψηφιακής Εγκληματολογίας

```
Εσείς: "Ανάλυσε τα suspect.png και suspect.wav για κρυφή επικοινωνία"

Agent: -> img_detect: Δεν ανιχνεύτηκε LSB στεγανογραφία, αλλά βρέθηκαν
          προσαρτημένα δεδομένα (12.847 bytes μετά τον δείκτη PNG IEND)
       -> file_identify: Τα προσαρτημένα δεδομένα αρχίζουν με υπογραφή ZIP (PK)
       -> img_metadata: Το PNG tEXt chunk περιέχει ασυνήθιστο κλειδί "comm"
          με τιμή Base64
       -> audio_detect: Το αρχείο WAV παρουσιάζει ανωμαλία LSB chi-square (p=0.003)
       -> audio_silence: Εντοπίστηκαν 3 σιωπηλές περιοχές με ενεργά LSBs
       -> audio_lsb_extract: Εξήχθησαν 2.048 bytes κρυμμένων δεδομένων

       "Και τα δύο αρχεία περιέχουν κρυμμένα δεδομένα:
        1. Το suspect.png έχει αρχείο ZIP προσαρτημένο μετά τον δείκτη EOF
           του PNG, συν ένα μήνυμα κωδικοποιημένο σε Base64 στα μεταδεδομένα
        2. Το suspect.wav περιέχει στεγανογραφία LSB, συγκεκριμένα σε
           σιωπηλές περιοχές — ισχυρή ένδειξη σκόπιμης απόκρυψης.
        Συνιστάται η εξαγωγή του ZIP και η ανάλυση του φορτίου WAV."
```

### Ανίχνευση Κειμένου Μηδενικού Πλάτους

```
Εσείς: "Αυτό το email φαίνεται φυσιολογικό αλλά μπορεί να έχει κρυφό κείμενο"

Agent: -> text_detect: Εντοπίστηκαν 847 χαρακτήρες μηδενικού πλάτους μεταξύ
          ορατών χαρακτήρων
       -> text_zwc_detect: ZWSP (342), ZWNJ (505) βρέθηκαν σε 127
          θέσεις σε όλο το κείμενο
       -> text_zwc_extract: Η δυαδική αποκωδικοποίηση αποδίδει "EXFIL-ROUTE-7"
       -> text_invisible_scan: 3 επιπλέον αόρατοι χαρακτήρες Unicode
          (U+2060, U+FEFF) βρέθηκαν στην ενότητα κεφαλίδας

       "Το email περιέχει κρυμμένο μήνυμα κωδικοποιημένο με χαρακτήρες
        μηδενικού πλάτους: 'EXFIL-ROUTE-7'. 847 αόρατοι χαρακτήρες είναι
        διασκορπισμένοι σε όλο το ορατό κείμενο χρησιμοποιώντας δυαδική
        κωδικοποίηση ZWSP/ZWNJ. Επιπλέον, βρέθηκαν 3 χαρακτήρες Word
        Joiner και BOM στην κεφαλίδα — πιθανώς τεχνουργήματα μεταδεδομένων."
```

### Εγκληματολογική Ανάλυση JPEG

```
Εσείς: "Είναι αυθεντική αυτή η εικόνα JPEG ή έχει παραποιηθεί;"

Agent: -> jpeg_structure: 14 δείκτες αναλύθηκαν, κανονική δομή JFIF
       -> jpeg_double_compression: Ανιχνεύτηκε διπλή συμπίεση —
          βαθμολογία τεχνουργημάτων blocks 0.73, αναλογία ανωμαλίας κβαντισμού 0.81
       -> jpeg_quantization: Ποιότητα ~75 (φωτεινότητα), αλλά υπολειπόμενα
          μοτίβα υποδεικνύουν αρχική ποιότητα ~92
       -> jpeg_exif_deep: Το EXIF αναφέρει "iPhone 14 Pro" αλλά το πεδίο
          Software αναφέρει "Adobe Photoshop 25.0"
       -> jpeg_thumbnail_compare: Οι διαστάσεις μικρογραφίας ΔΕΝ ταιριάζουν
          με την κύρια εικόνα (μικρογραφία: 4032x3024, κύρια: 2048x1536)

       "Αυτό το JPEG έχει παραποιηθεί. Στοιχεία:
        1. Ανιχνεύτηκε διπλή συμπίεση — η εικόνα αποκωδικοποιήθηκε και
           επανακωδικοποιήθηκε σε χαμηλότερη ποιότητα (αρχική ~92, τρέχουσα ~75)
        2. Το EXIF δηλώνει iPhone 14 Pro αλλά το πεδίο λογισμικού αποκαλύπτει
           επεξεργασία Photoshop
        3. Η μικρογραφία είναι από την αρχική λήψη 4032x3024 αλλά η κύρια
           εικόνα μεταβλήθηκε σε 2048x1536
        Και τα τρία ευρήματα επιβεβαιώνουν ανεξάρτητα τροποποίηση μετά
        τη λήψη."
```

## Χρήση CLI

```bash
# Εμφάνιση βοήθειας
npx -y steganography-mcp --help

# Λίστα και των 60 εργαλείων με περιγραφές
npx -y steganography-mcp --list

# Ανίχνευση στεγανογραφίας σε εικόνα
npx -y steganography-mcp --tool img_detect '{"file_path":"challenge.png"}'

# Εξαγωγή κρυμμένου μηνύματος από LSBs
npx -y steganography-mcp --tool img_lsb_extract '{"file_path":"stego.png"}'

# Στεγανάλυση chi-square
npx -y steganography-mcp --tool img_chi_square '{"file_path":"suspect.png"}'

# Ανάλυση RS (μέθοδος Fridrich-Goljan-Du)
npx -y steganography-mcp --tool img_rs_analysis '{"file_path":"suspect.png"}'

# Ανίχνευση διπλής συμπίεσης JPEG
npx -y steganography-mcp --tool jpeg_double_compression '{"file_path":"photo.jpg"}'

# Βαθιά ανάλυση EXIF
npx -y steganography-mcp --tool jpeg_exif_deep '{"file_path":"photo.jpg"}'

# Ανίχνευση στεγανογραφίας ήχου
npx -y steganography-mcp --tool audio_detect '{"file_path":"message.wav"}'

# Ανίχνευση κωδικοποίησης χαρακτήρων μηδενικού πλάτους
npx -y steganography-mcp --tool text_zwc_detect '{"text":"suspicious text here"}'

# Ενσωμάτωση κρυμμένου μηνύματος με χαρακτήρες μηδενικού πλάτους
npx -y steganography-mcp --tool text_zwc_embed '{"text":"cover text","message":"secret"}'

# Αναγνώριση τύπου αρχείου και ανίχνευση πολύγλωσσων
npx -y steganography-mcp --tool file_polyglot '{"file_path":"suspicious.pdf"}'

# Σάρωση για ενσωματωμένα αρχεία (σε στυλ binwalk)
npx -y steganography-mcp --tool file_embedded '{"file_path":"mystery.bin"}'

# Οπτικοποίηση εντροπίας
npx -y steganography-mcp --tool file_entropy_visual '{"file_path":"data.bin"}'

# Αυτόματη ανίχνευση κωδικοποίησης
npx -y steganography-mcp --tool crypto_detect '{"input":"aGVsbG8gd29ybGQ="}'

# XOR brute-force
npx -y steganography-mcp --tool crypto_xor '{"input":"4f5243484e"}'

# Ανίχνευση μοτίβων κρυπτογράφησης
npx -y steganography-mcp --tool crypto_patterns '{"input":"Gur dhvpx oebja sbk"}'

# Χρήση Bun (ταχύτερη εκκίνηση)
bunx steganography-mcp --tool img_detect '{"file_path":"image.png"}'
```

## Περιπτώσεις Χρήσης

### Προκλήσεις CTF
Λύστε προκλήσεις στεγανογραφίας σε διαγωνισμούς capture-the-flag. Ο AI agent μπορεί να εφαρμόσει συστηματικά όλες τις τεχνικές ανίχνευσης &mdash; ανάλυση LSB, επιθεώρηση μεταδεδομένων, προσαρτημένα δεδομένα, ανίχνευση κωδικοποίησης και αναγνώριση κρυπτογράφησης &mdash; για να βρει κρυμμένες σημαίες σε εικόνες, αρχεία ήχου, έγγραφα και κείμενο.

### Ψηφιακή Εγκληματολογία
Ανιχνεύστε κρυφά κανάλια επικοινωνίας σε εγκληματολογικές έρευνες. Αναλύστε ύποπτα αρχεία για κρυμμένα δεδομένα χρησιμοποιώντας στατιστική στεγανάλυση (chi-square, ανάλυση RS), ελέγξτε για δεδομένα προσαρτημένα μετά τους δείκτες EOF, σαρώστε για ενσωματωμένα αρχεία και αναγνωρίστε υπογραφές εργαλείων στεγανογραφίας.

### Έρευνα Ασφάλειας
Αναλύστε εργαλεία και τεχνικές στεγανογραφίας. Συγκρίνετε αρχικές και στεγανογραφημένες εικόνες pixel-προς-pixel, μελετήστε κατανομές συντελεστών DCT σε JPEG στεγανογραφία, μετρήστε αλλαγές εντροπίας από ενσωμάτωση και αντίστροφα μηχανικά κωδικοποιήσεις.

### Εκπαίδευση
Μάθετε πώς λειτουργούν οι τεχνικές στεγανογραφίας. Ενσωματώστε και εξάγετε μηνύματα LSB, κωδικοποιήστε κείμενο με χαρακτήρες μηδενικού πλάτους, οπτικοποιήστε bit planes και χάρτες εντροπίας, αναλύστε δομές αρχείων με hex dumps και μελετήστε μοτίβα κρυπτογράφησης με ανάλυση συχνοτήτων.

### Απόκριση σε Περιστατικά
Κατά την απόκριση σε περιστατικά, ελέγξτε έγγραφα και εικόνες για κρυφά κανάλια εξαγωγής δεδομένων. Σαρώστε PDFs για κρυφό JavaScript και ενσωματωμένα αρχεία, ανιχνεύστε κωδικοποίηση χαρακτήρων μηδενικού πλάτους σε emails, αναγνωρίστε πολύγλωσσα αρχεία και αναλύστε ύποπτες κωδικοποιήσεις.

## Αρχιτεκτονική

```
src/
  index.ts                    # Σημείο εισόδου CLI (--help, --list, --tool, stdio server)
  protocol/
    mcp-server.ts             # Ρύθμιση MCP server (μεταφορά stdio)
    tools.ts                  # Μητρώο εργαλείων — και τα 60 εργαλεία συγκεντρωμένα εδώ
  types/
    index.ts                  # Κοινοί τύποι (ToolDef, ToolContext, ToolResult)
  utils/
    binary.ts                 # Ανάγνωση δυαδικών αρχείων, hex dump, ανίχνευση μορφής
    stats.ts                  # Εντροπία Shannon, chi-square, συχνότητα bytes
    cache.ts                  # Κρυφή μνήμη TTL
    png-parser.ts             # Αμιγής TS αναλυτής PNG (IHDR, chunks, δεδομένα pixel)
    jpeg-parser.ts            # Αμιγής TS αναλυτής JPEG (δείκτες, EXIF, κβαντισμός)
    wav-parser.ts             # Αμιγής TS αναλυτής WAV (chunks RIFF, δείγματα PCM)
    bmp-parser.ts             # Αμιγής TS αναλυτής BMP (κεφαλίδα, δεδομένα pixel)
  image/                      # Εργαλεία Στεγανάλυσης Εικόνας (14)
  jpeg/                       # Εργαλεία Ανάλυσης JPEG (7)
  audio/                      # Εργαλεία Στεγανάλυσης Ήχου (7)
  text/                       # Εργαλεία Κειμένου & Unicode (10)
  file/                       # Εργαλεία Εγκληματολογίας Αρχείων (10)
  document/                   # Εργαλεία Ανάλυσης Εγγράφων (5)
  crypto/                     # Εργαλεία Κωδικοποίησης & Κρυπτογραφίας (7)
  data/
    encoding-patterns.ts      # Μοτίβα regex κωδικοποίησης + αποκωδικοποιητές
    magic-bytes.ts            # Βάση δεδομένων υπογραφών αρχείων (100+ μορφές)
    stego-signatures.ts       # Γνωστές υπογραφές εργαλείων στεγανογραφίας
    unicode-invisible.ts      # Βάση δεδομένων αόρατων χαρακτήρων Unicode
```

**Σχεδιαστικές αποφάσεις:**

- **4 εξαρτήσεις, τίποτα άλλο** &mdash; `@modelcontextprotocol/sdk` για το πρωτόκολλο MCP, `zod` για επικύρωση εισόδου, `pngjs` για πρόσβαση pixel PNG, `jpeg-js` για αποκωδικοποίηση JPEG. Χωρίς φουσκωμένο δέντρο εξαρτήσεων. Χωρίς native modules. Χωρίς C bindings. Χωρίς Python. Χωρίς Java.
- **100% εκτός σύνδεσης** &mdash; Κάθε εργαλείο εκτελείται εξ ολοκλήρου τοπικά. Χωρίς HTTP αιτήματα. Χωρίς API κλήσεις. Χωρίς τηλεμετρία. Χωρίς εξαρτήσεις cloud. Τα αρχεία σας δεν φεύγουν ποτέ από τον υπολογιστή σας.
- **Αμιγής TypeScript στατιστική ανάλυση** &mdash; Δοκιμή chi-square, ανάλυση RS (Fridrich-Goljan-Du), Ανάλυση Ζευγών Δειγμάτων, εντροπία Shannon, Δείκτης Σύμπτωσης και ανάλυση συχνοτήτων υλοποιούνται πλήρως σε αμιγή TypeScript. Χωρίς εξωτερικές βιβλιοθήκες μαθηματικών.
- **Προσαρμοσμένοι αναλυτές μορφών** &mdash; PNG chunks, δείκτες JPEG/EXIF/πίνακες κβαντισμού, RIFF chunks WAV και κεφαλίδες BMP αναλύονται χωρίς εξωτερικές εξαρτήσεις χρησιμοποιώντας τους αναλυτές `utils/`. Αυτό επιτρέπει βαθιά ανάλυση ειδική ανά μορφή που οι βιβλιοθήκες γενικού σκοπού δεν μπορούν να παρέχουν.
- **7 πάροχοι, 1 server** &mdash; Κάθε κατηγορία ανάλυσης είναι ανεξάρτητη μονάδα. Ο AI agent επιλέγει ποια εργαλεία θα χρησιμοποιήσει βάσει του πλαισίου της έρευνας.
- **Καθαρό μοτίβο ToolDef** &mdash; Κάθε εργαλείο ακολουθεί το ίδιο μοτίβο `{ name, description, schema, execute }`. Η προσθήκη νέου εργαλείου είναι ένα αντικείμενο στη σχετική μονάδα.
- **Επικύρωση Zod σε κάθε πεδίο** &mdash; Κάθε πεδίο schema έχει `.describe()` για πλαίσιο AI agent. Μη έγκυρες εισαγωγές εντοπίζονται πριν την εκτέλεση με σαφή μηνύματα σφάλματος.
