## সমস্যা

স্টেগানোগ্রাফি হলো প্রকাশ্যে ডেটা লুকানোর শিল্প &mdash; ছবি, অডিও ফাইল, ডকুমেন্ট এবং এমনকি ইউনিকোড টেক্সটের ভেতরে। এটি CTF প্রতিযোগিতা, ডিজিটাল ফরেনসিক তদন্ত, গোপন যোগাযোগ চ্যানেল এবং ম্যালওয়্যার পেলোডে ব্যবহৃত হয়। এটি সনাক্ত করতে পরিসংখ্যানগত বিশ্লেষণ, ফরম্যাট-নির্দিষ্ট পার্সিং, এন্ট্রপি পরিমাপ এবং বিশেষজ্ঞ জ্ঞানের সমন্বয় প্রয়োজন।

```
প্রচলিত স্টেগানোগ্রাফি বিশ্লেষণ কর্মপ্রবাহ:
  ইমেজ স্টেগো সনাক্তকরণ        ->  zsteg + stegsolve (২টি টুল, Ruby + Java)
  কাই-স্কোয়ার বিশ্লেষণ          ->  কাস্টম Python স্ক্রিপ্ট
  RS বিশ্লেষণ                    ->  কাস্টম MATLAB/Python কোড
  JPEG DCT ফরেনসিক্স             ->  stegdetect (পরিত্যক্ত C টুল, ২০০৪ থেকে)
  LSB ডেটা এক্সট্র্যাক্ট         ->  zsteg + steghide + openstego (৩টি টুল)
  অডিও স্টেগানোগ্রাফি           ->  Audacity ম্যানুয়াল + কাস্টম স্ক্রিপ্ট
  জিরো-উইডথ টেক্সট সনাক্তকরণ    ->  ওয়েব-ভিত্তিক টুল + ম্যানুয়াল পরীক্ষা
  ফাইল ফরেনসিক্স / binwalk      ->  binwalk + foremost + xxd (৩টি টুল)
  EXIF মেটাডেটা                  ->  exiftool (Perl ডিপেন্ডেন্সি)
  এনকোডিং সনাক্তকরণ             ->  CyberChef ওয়েব UI + ম্যানুয়াল অনুমান
  ─────────────────────────────────
  মোট: ১০+ টুল, ৫+ ভাষা, ঘণ্টার পর ঘণ্টা ম্যানুয়াল সমন্বয়
```

**steganography-mcp** আপনার AI এজেন্টকে [Model Context Protocol](https://modelcontextprotocol.io)-এর মাধ্যমে ১৭টি বিভাগে ১২৮টি টুল প্রদান করে। এজেন্ট ইমেজ স্টেগ্যানালাইসিস, JPEG ফরেনসিক্স, অডিও বিশ্লেষণ, টেক্সট স্টেগানোগ্রাফি সনাক্তকরণ, ফাইল ফরেনসিক্স, ডকুমেন্ট বিশ্লেষণ, এনকোডিং শনাক্তকরণ, ভিডিও নেটওয়ার্ক ও আর্কাইভ বিশ্লেষণ, GIF ও MP3 স্টেগানোগ্রাফি, স্প্রেড স্পেকট্রাম ও BPCS বিশ্লেষণ, এবং তৈরি ও এম্বেডিং করতে পারে &mdash; সবকিছু একটি মাত্র কথোপকথনে, ১০০% স্থানীয়ভাবে চলে, বাহ্যিক সেবার উপর কোনো নির্ভরতা ছাড়াই।

```
steganography-mcp দিয়ে:
  আপনি: "এই CTF চ্যালেঞ্জ ইমেজটি লুকানো ডেটার জন্য বিশ্লেষণ করুন"

  এজেন্ট: -> img_detect: Chi-square p=0.0001 (LSB এম্বেডিং সনাক্ত),
            RS বিশ্লেষণ ৪২% এম্বেডিং রেট অনুমান করেছে, নিচের-ডান
            কোয়াড্র্যান্টে এন্ট্রপি অস্বাভাবিকতা
         -> img_lsb_extract: RGB LSB থেকে ৮৪৭ বাইট এক্সট্র্যাক্ট করা হয়েছে
         -> crypto_detect: এক্সট্র্যাক্ট করা ডেটা Base64-এনকোডেড
         -> crypto_decode: ডিকোড করা হয়েছে "FLAG{hidden_in_plain_sight_2024}"
         -> img_known_tools: OpenStego-এর সিগনেচার ম্যাচ

         "ইমেজটিতে OpenStego দিয়ে এম্বেড করা LSB স্টেগানোগ্রাফি আছে।
          কাই-স্কোয়ার টেস্ট তিনটি RGB চ্যানেলেই LSB প্রতিস্থাপন
          নিশ্চিত করেছে ৪২% এম্বেডিং রেটে। লুকানো পেলোড
          Base64-এনকোডেড এবং ফ্ল্যাগে ডিকোড হয়:
          FLAG{hidden_in_plain_sight_2024}"
```

## দ্রুত শুরু

### বিকল্প ১: npx (ইনস্টল ছাড়াই)

```bash
npx -y steganography-mcp
```

সমস্ত ১২৮টি টুল তাৎক্ষণিকভাবে কাজ করে। কোনো API কী নেই। কোনো কনফিগারেশন নেই। ১০০% অফলাইন।

### বিকল্প ২: bunx (দ্রুততর)

```bash
bunx steganography-mcp
```

### বিকল্প ৩: ক্লোন

```bash
git clone https://github.com/badchars/steganography-mcp.git
cd steganography-mcp
bun install
```

</details>

```json
      "command": "npx",

  }
```

</details>

<details>

</details>

## AI কী করতে পারে

### CTF চ্যালেঞ্জ সমাধানকারী

```
আপনি: "এই CTF স্টেগানোগ্রাফি চ্যালেঞ্জটি সমাধান করুন — ইমেজটি challenge.png-এ আছে"

এজেন্ট: -> img_detect: কাই-স্কোয়ার স্টেগ্যানালাইসিস LSB এম্বেডিং সনাক্ত করেছে
          (p=0.00012), RS বিশ্লেষণ ৩৮% এম্বেডিং রেট অনুমান করেছে,
          ব্লক (12,4)-(15,8)-এ এন্ট্রপি অস্বাভাবিকতা
       -> img_lsb_extract: RGB চ্যানেল LSB থেকে ১,২৪৭ বাইট এক্সট্র্যাক্ট করা হয়েছে
       -> crypto_detect: ডেটা Base64 প্যাটার্নের সাথে মিলেছে (কনফিডেন্স: ০.৯৫)
       -> crypto_decode: "Q1RGe3N0M2cwX21hc3Rlcn0=" -> "CTF{st3g0_master}"
       -> img_known_tools: PNG চাংকে OpenStego সিগনেচার সনাক্ত হয়েছে

       "ফ্ল্যাগ পাওয়া গেছে: CTF{st3g0_master}
        ইমেজটি OpenStego দিয়ে এম্বেড করা LSB স্টেগানোগ্রাফি ব্যবহার করে।
        পেলোড RGB-এর সবচেয়ে কম গুরুত্বপূর্ণ বিটে Base64-এনকোডেড
        ছিল ৩৮% এম্বেডিং রেটে।"
```

### ডিজিটাল ফরেনসিক তদন্ত

```
আপনি: "suspect.png এবং suspect.wav গোপন যোগাযোগের জন্য বিশ্লেষণ করুন"

এজেন্ট: -> img_detect: কোনো LSB স্টেগো সনাক্ত হয়নি, তবে সংযুক্ত ডেটা পাওয়া গেছে
          (PNG IEND মার্কারের পরে ১২,৮৪৭ বাইট)
       -> file_identify: সংযুক্ত ডেটা ZIP সিগনেচার (PK) দিয়ে শুরু হয়
       -> img_metadata: PNG tEXt চাংকে অস্বাভাবিক কী "comm" আছে
          Base64 মান সহ
       -> audio_detect: WAV ফাইলে LSB কাই-স্কোয়ার অস্বাভাবিকতা (p=0.003)
       -> audio_silence: ৩টি নীরব অংশে সক্রিয় LSB সনাক্ত হয়েছে
       -> audio_lsb_extract: ২,০৪৮ বাইট লুকানো ডেটা এক্সট্র্যাক্ট করা হয়েছে

       "উভয় ফাইলেই লুকানো ডেটা রয়েছে:
        ১. suspect.png-এ PNG EOF মার্কারের পরে একটি ZIP আর্কাইভ
           সংযুক্ত আছে, সাথে মেটাডেটায় Base64-এনকোডেড বার্তা
        ২. suspect.wav-এ LSB স্টেগানোগ্রাফি আছে, বিশেষত নীরব
           অংশে — ইচ্ছাকৃত লুকানোর একটি শক্তিশালী নির্দেশক।
        ZIP এক্সট্র্যাক্ট করা এবং WAV পেলোড বিশ্লেষণ করার পরামর্শ দেওয়া হচ্ছে।"
```

### জিরো-উইডথ টেক্সট সনাক্তকরণ

```
আপনি: "এই ইমেইলটি স্বাভাবিক দেখাচ্ছে কিন্তু লুকানো টেক্সট থাকতে পারে"

এজেন্ট: -> text_detect: দৃশ্যমান অক্ষরের মধ্যে ৮৪৭টি জিরো-উইডথ
          ক্যারেক্টার সনাক্ত হয়েছে
       -> text_zwc_detect: ZWSP (৩৪২), ZWNJ (৫০৫) টেক্সট জুড়ে
          ১২৭টি অবস্থানে পাওয়া গেছে
       -> text_zwc_extract: বাইনারি ডিকোডিং দিয়েছে "EXFIL-ROUTE-7"
       -> text_invisible_scan: হেডার অংশে আরো ৩টি অদৃশ্য ইউনিকোড
          ক্যারেক্টার (U+2060, U+FEFF) পাওয়া গেছে

       "ইমেইলটিতে জিরো-উইডথ ক্যারেক্টার দিয়ে এনকোডেড একটি
        লুকানো বার্তা আছে: 'EXFIL-ROUTE-7'। ৮৪৭টি অদৃশ্য ক্যারেক্টার
        ZWSP/ZWNJ বাইনারি এনকোডিং ব্যবহার করে দৃশ্যমান টেক্সট জুড়ে
        ছড়িয়ে আছে। এছাড়াও, হেডারে ৩টি Word Joiner এবং BOM
        ক্যারেক্টার পাওয়া গেছে — সম্ভবত মেটাডেটা আর্টিফ্যাক্ট।"
```

### JPEG ফরেনসিক বিশ্লেষণ

```
আপনি: "এই JPEG ইমেজটি আসল নাকি টেম্পার করা হয়েছে?"

এজেন্ট: -> jpeg_structure: ১৪টি মার্কার পার্স করা হয়েছে, স্বাভাবিক JFIF গঠন
       -> jpeg_double_compression: ডাবল কম্প্রেশন সনাক্ত —
          ব্লকিং আর্টিফ্যাক্ট স্কোর ০.৭৩, কোয়ান্টাইজেশন অস্বাভাবিকতা অনুপাত ০.৮১
       -> jpeg_quantization: গুণমান ~৭৫ (লুমিন্যান্স), কিন্তু অবশিষ্ট
          প্যাটার্ন দেখাচ্ছে মূল ছিল গুণমান ~৯২
       -> jpeg_exif_deep: EXIF বলছে "iPhone 14 Pro" কিন্তু Software
          ফিল্ড বলছে "Adobe Photoshop 25.0"
       -> jpeg_thumbnail_compare: থাম্বনেইলের মাত্রা মূল ইমেজের সাথে
          মিলছে না (থাম্বনেইল: 4032x3024, মূল: 2048x1536)

       "এই JPEG-এ টেম্পারিং হয়েছে। প্রমাণ:
        ১. ডাবল কম্প্রেশন সনাক্ত — ইমেজটি ডিকোড করে কম
           গুণমানে পুনরায় এনকোড করা হয়েছে (মূল ~৯২, বর্তমান ~৭৫)
        ২. EXIF iPhone 14 Pro দাবি করছে কিন্তু সফটওয়্যার ফিল্ড
           Photoshop এডিটিং প্রকাশ করছে
        ৩. থাম্বনেইল মূল 4032x3024 ক্যাপচার থেকে কিন্তু
           মূল ইমেজ 2048x1536-এ রিসাইজ করা হয়েছে
        তিনটি ফলাফলই স্বাধীনভাবে ক্যাপচার-পরবর্তী
        পরিবর্তন নিশ্চিত করছে।"
```

| `file_strings` | বাইনারি ফাইল থেকে মুদ্রণযোগ্য এবং ইউনিকোড স্ট্রিং এক্সট্র্যাক্ট করে। মুদ্রণযোগ্য ক্যারেক্টারের রান স্ক্যান করে এবং ফাইল অফসেট সহ রিপোর্ট করে। ASCII, UTF-8, UTF-16 সমর্থন করে |

## CLI ব্যবহার

```bash
# সাহায্য দেখান
npx -y steganography-mcp --help

# বিবরণ সহ সমস্ত ১২৮টি টুলের তালিকা
npx -y steganography-mcp --list

# ইমেজে স্টেগানোগ্রাফি সনাক্ত করুন
npx -y steganography-mcp --tool img_detect '{"file_path":"challenge.png"}'

# LSB থেকে লুকানো বার্তা এক্সট্র্যাক্ট করুন
npx -y steganography-mcp --tool img_lsb_extract '{"file_path":"stego.png"}'

# কাই-স্কোয়ার স্টেগ্যানালাইসিস
npx -y steganography-mcp --tool img_chi_square '{"file_path":"suspect.png"}'

# RS বিশ্লেষণ (Fridrich-Goljan-Du পদ্ধতি)
npx -y steganography-mcp --tool img_rs_analysis '{"file_path":"suspect.png"}'

# JPEG ডাবল কম্প্রেশন সনাক্তকরণ
npx -y steganography-mcp --tool jpeg_double_compression '{"file_path":"photo.jpg"}'

# গভীর EXIF বিশ্লেষণ
npx -y steganography-mcp --tool jpeg_exif_deep '{"file_path":"photo.jpg"}'

# অডিও স্টেগানোগ্রাফি সনাক্তকরণ
npx -y steganography-mcp --tool audio_detect '{"file_path":"message.wav"}'

# জিরো-উইডথ ক্যারেক্টার এনকোডিং সনাক্ত করুন
npx -y steganography-mcp --tool text_zwc_detect '{"text":"suspicious text here"}'

# জিরো-উইডথ ক্যারেক্টার দিয়ে লুকানো বার্তা এম্বেড করুন
npx -y steganography-mcp --tool text_zwc_embed '{"text":"cover text","message":"secret"}'

# ফাইল টাইপ শনাক্ত করুন এবং পলিগ্লট সনাক্ত করুন
npx -y steganography-mcp --tool file_polyglot '{"file_path":"suspicious.pdf"}'

# এম্বেডেড ফাইল স্ক্যান করুন (binwalk-স্টাইলে)
npx -y steganography-mcp --tool file_embedded '{"file_path":"mystery.bin"}'

# এন্ট্রপি ভিজুয়ালাইজেশন
npx -y steganography-mcp --tool file_entropy_visual '{"file_path":"data.bin"}'

# স্বয়ংক্রিয় এনকোডিং সনাক্তকরণ
npx -y steganography-mcp --tool crypto_detect '{"input":"aGVsbG8gd29ybGQ="}'

# XOR ব্রুট-ফোর্স
npx -y steganography-mcp --tool crypto_xor '{"input":"4f5243484e"}'

# সাইফার প্যাটার্ন সনাক্ত করুন
npx -y steganography-mcp --tool crypto_patterns '{"input":"Gur dhvpx oebja sbk"}'

# Bun ব্যবহার করুন (দ্রুত স্টার্টআপ)
bunx steganography-mcp --tool img_detect '{"file_path":"image.png"}'
```

## ব্যবহারের ক্ষেত্র

### CTF চ্যালেঞ্জ
ক্যাপচার-দ্য-ফ্ল্যাগ প্রতিযোগিতায় স্টেগানোগ্রাফি চ্যালেঞ্জ সমাধান করুন। AI এজেন্ট পদ্ধতিগতভাবে সমস্ত সনাক্তকরণ কৌশল প্রয়োগ করতে পারে &mdash; LSB বিশ্লেষণ, মেটাডেটা পরীক্ষা, সংযুক্ত ডেটা, এনকোডিং সনাক্তকরণ এবং সাইফার শনাক্তকরণ &mdash; ইমেজ, অডিও ফাইল, ডকুমেন্ট এবং টেক্সটে লুকানো ফ্ল্যাগ খুঁজে পেতে।

### ডিজিটাল ফরেনসিক্স
ফরেনসিক তদন্তে গোপন যোগাযোগ চ্যানেল সনাক্ত করুন। পরিসংখ্যানগত স্টেগ্যানালাইসিস (কাই-স্কোয়ার, RS বিশ্লেষণ) ব্যবহার করে সন্দেহভাজন ফাইলে লুকানো ডেটা বিশ্লেষণ করুন, EOF মার্কারের পরে সংযুক্ত ডেটা পরীক্ষা করুন, এম্বেডেড ফাইল স্ক্যান করুন এবং স্টেগানোগ্রাফি টুল সিগনেচার শনাক্ত করুন।

### নিরাপত্তা গবেষণা
স্টেগানোগ্রাফি টুল এবং কৌশল বিশ্লেষণ করুন। মূল এবং স্টেগো ইমেজ পিক্সেল-বাই-পিক্সেল তুলনা করুন, JPEG স্টেগোতে DCT কোএফিসিয়েন্ট বিতরণ অধ্যয়ন করুন, এম্বেডিং থেকে এন্ট্রপি পরিবর্তন পরিমাপ করুন এবং এনকোডিং স্কিম রিভার্স-ইঞ্জিনিয়ার করুন।

### শিক্ষা
স্টেগানোগ্রাফি কৌশল কীভাবে কাজ করে শিখুন। LSB বার্তা এম্বেড ও এক্সট্র্যাক্ট করুন, জিরো-উইডথ ক্যারেক্টার দিয়ে টেক্সট এনকোড করুন, বিট প্লেন ও এন্ট্রপি ম্যাপ ভিজুয়ালাইজ করুন, হেক্স ডাম্প দিয়ে ফাইল গঠন বিশ্লেষণ করুন এবং ফ্রিকোয়েন্সি বিশ্লেষণ দিয়ে সাইফার প্যাটার্ন অধ্যয়ন করুন।

### ঘটনা প্রতিক্রিয়া
ঘটনা প্রতিক্রিয়ার সময়, লুকানো ডেটা এক্সফিলট্রেশন চ্যানেলের জন্য ডকুমেন্ট এবং ইমেজ পরীক্ষা করুন। লুকানো JavaScript এবং এম্বেডেড ফাইলের জন্য PDF স্ক্যান করুন, ইমেইলে জিরো-উইডথ ক্যারেক্টার এনকোডিং সনাক্ত করুন, পলিগ্লট ফাইল শনাক্ত করুন এবং সন্দেহজনক এনকোডিং বিশ্লেষণ করুন।

**ডিজাইন সিদ্ধান্তসমূহ:**

- **৪টি ডিপেন্ডেন্সি, এর বেশি নয়** &mdash; MCP প্রোটোকলের জন্য `@modelcontextprotocol/sdk`, ইনপুট ভ্যালিডেশনের জন্য `zod`, PNG পিক্সেল অ্যাক্সেসের জন্য `pngjs`, JPEG ডিকোডিংয়ের জন্য `jpeg-js`। কোনো ফোলা ডিপেন্ডেন্সি ট্রি নেই। কোনো নেটিভ মডিউল নেই। কোনো C বাইন্ডিং নেই। কোনো Python নেই। কোনো Java নেই।
- **১০০% অফলাইন** &mdash; প্রতিটি টুল সম্পূর্ণ স্থানীয়ভাবে চলে। কোনো HTTP রিকোয়েস্ট নেই। কোনো API কল নেই। কোনো টেলিমেট্রি নেই। কোনো ক্লাউড ডিপেন্ডেন্সি নেই। আপনার ফাইল কখনো আপনার মেশিন ছেড়ে যায় না।
- **বিশুদ্ধ TypeScript পরিসংখ্যানগত বিশ্লেষণ** &mdash; কাই-স্কোয়ার টেস্ট, RS বিশ্লেষণ (Fridrich-Goljan-Du), স্যাম্পল পেয়ার অ্যানালাইসিস, শ্যানন এন্ট্রপি, Index of Coincidence এবং ফ্রিকোয়েন্সি বিশ্লেষণ সবই বিশুদ্ধ TypeScript-এ বাস্তবায়িত। কোনো বাহ্যিক গণিত লাইব্রেরি নেই।
- **কাস্টম ফরম্যাট পার্সার** &mdash; PNG চাংক, JPEG মার্কার/EXIF/কোয়ান্টাইজেশন টেবিল, WAV RIFF চাংক এবং BMP হেডার `utils/` পার্সার ব্যবহার করে শূন্য বাহ্যিক ডিপেন্ডেন্সিতে পার্স করা হয়। এটি সাধারণ-উদ্দেশ্যের লাইব্রেরি যা প্রদান করতে পারে না এমন গভীর ফরম্যাট-নির্দিষ্ট বিশ্লেষণের সুযোগ দেয়।
- **১৭টি প্রোভাইডার, ১টি সার্ভার** &mdash; প্রতিটি বিশ্লেষণ বিভাগ একটি স্বাধীন মডিউল। AI এজেন্ট তদন্তের প্রসঙ্গ অনুসারে কোন টুল ব্যবহার করবে তা বেছে নেয়।
- **পরিচ্ছন্ন ToolDef প্যাটার্ন** &mdash; প্রতিটি টুল একই `{ name, description, schema, execute }` প্যাটার্ন অনুসরণ করে। নতুন টুল যোগ করতে উপযুক্ত মডিউলে একটি মাত্র অবজেক্ট।
- **প্রতিটি ফিল্ডে Zod ভ্যালিডেশন** &mdash; প্রতিটি স্কিমা ফিল্ডে AI এজেন্ট প্রসঙ্গের জন্য `.describe()` আছে। অবৈধ ইনপুট স্পষ্ট ত্রুটি বার্তা সহ এক্সিকিউশনের আগেই ধরা পড়ে।
