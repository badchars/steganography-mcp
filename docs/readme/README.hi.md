## समस्या

स्टेगनोग्राफी डेटा को सबकी नज़रों के सामने छिपाने की कला है &mdash; छवियों, ऑडियो फ़ाइलों, दस्तावेज़ों और यहाँ तक कि यूनिकोड टेक्स्ट के अंदर। इसका उपयोग CTF प्रतियोगिताओं, डिजिटल फोरेंसिक जाँच, गुप्त संचार चैनलों और मैलवेयर पेलोड में किया जाता है। इसका पता लगाने के लिए सांख्यिकीय विश्लेषण, फ़ॉर्मेट-विशिष्ट पार्सिंग, एन्ट्रॉपी मापन और विशेषज्ञ ज्ञान के संयोजन की आवश्यकता होती है।

```
पारंपरिक स्टेगनोग्राफी विश्लेषण कार्यप्रवाह:
  इमेज स्टेगो पहचान             ->  zsteg + stegsolve (2 टूल, Ruby + Java)
  काई-स्क्वेयर विश्लेषण          ->  कस्टम Python स्क्रिप्ट
  RS विश्लेषण                    ->  कस्टम MATLAB/Python कोड
  JPEG DCT फोरेंसिक्स             ->  stegdetect (परित्यक्त C टूल, 2004 से)
  LSB डेटा निकालना               ->  zsteg + steghide + openstego (3 टूल)
  ऑडियो स्टेगनोग्राफी            ->  Audacity मैनुअल + कस्टम स्क्रिप्ट
  ज़ीरो-विड्थ टेक्स्ट पहचान       ->  वेब-आधारित टूल + मैनुअल निरीक्षण
  फ़ाइल फोरेंसिक्स / binwalk     ->  binwalk + foremost + xxd (3 टूल)
  EXIF मेटाडेटा                  ->  exiftool (Perl डिपेंडेंसी)
  एनकोडिंग पहचान                 ->  CyberChef वेब UI + मैनुअल अनुमान
  ─────────────────────────────────
  कुल: 10+ टूल, 5+ भाषाएँ, घंटों का मैनुअल समन्वय
```

**steganography-mcp** आपके AI एजेंट को [Model Context Protocol](https://modelcontextprotocol.io) के माध्यम से 17 श्रेणियों में 128 टूल प्रदान करता है। एजेंट इमेज स्टेगैनालिसिस, JPEG फोरेंसिक्स, ऑडियो विश्लेषण, टेक्स्ट स्टेगनोग्राफी पहचान, फ़ाइल फोरेंसिक्स, दस्तावेज़ विश्लेषण, एनकोडिंग शनाख्त, वीडियो नेटवर्क और आर्काइव विश्लेषण, GIF और MP3 स्टेगनोग्राफी, स्प्रेड स्पेक्ट्रम और BPCS विश्लेषण, तथा निर्माण और एम्बेडिंग करता है &mdash; सब कुछ एक ही बातचीत में, 100% स्थानीय रूप से चलता है, बाहरी सेवाओं पर कोई निर्भरता नहीं।

```
steganography-mcp के साथ:
  आप: "इस CTF चैलेंज इमेज को छिपे हुए डेटा के लिए विश्लेषण करें"

  एजेंट: -> img_detect: Chi-square p=0.0001 (LSB एम्बेडिंग पाई गई),
            RS विश्लेषण 42% एम्बेडिंग दर का अनुमान लगाता है,
            निचले-दाएँ चतुर्थांश में एन्ट्रॉपी विसंगति
         -> img_lsb_extract: RGB LSB से 847 बाइट निकाले गए
         -> crypto_detect: निकाला गया डेटा Base64-एनकोडेड है
         -> crypto_decode: डिकोड किया गया "FLAG{hidden_in_plain_sight_2024}"
         -> img_known_tools: OpenStego का सिग्नेचर मिला

         "इमेज में OpenStego द्वारा एम्बेड की गई LSB स्टेगनोग्राफी है।
          काई-स्क्वेयर टेस्ट तीनों RGB चैनलों में LSB प्रतिस्थापन की
          पुष्टि करता है 42% एम्बेडिंग दर के साथ। छिपा हुआ पेलोड
          Base64-एनकोडेड है और फ़्लैग में डिकोड होता है:
          FLAG{hidden_in_plain_sight_2024}"
```

## AI क्या कर सकता है

### CTF चैलेंज सॉल्वर

```
आप: "इस CTF स्टेगनोग्राफी चैलेंज को हल करें — इमेज challenge.png पर है"

एजेंट: -> img_detect: काई-स्क्वेयर स्टेगैनालिसिस ने LSB एम्बेडिंग पाई
          (p=0.00012), RS विश्लेषण 38% एम्बेडिंग दर का अनुमान लगाता है,
          ब्लॉक (12,4)-(15,8) पर एन्ट्रॉपी विसंगति
       -> img_lsb_extract: RGB चैनल LSB से 1,247 बाइट निकाले गए
       -> crypto_detect: डेटा Base64 पैटर्न से मेल खाता है (विश्वास: 0.95)
       -> crypto_decode: "Q1RGe3N0M2cwX21hc3Rlcn0=" -> "CTF{st3g0_master}"
       -> img_known_tools: PNG चंक में OpenStego सिग्नेचर पाया गया

       "फ़्लैग मिला: CTF{st3g0_master}
        इमेज OpenStego द्वारा एम्बेड की गई LSB स्टेगनोग्राफी का उपयोग
        करती है। पेलोड RGB के सबसे कम महत्वपूर्ण बिट में
        Base64-एनकोडेड था 38% एम्बेडिंग दर के साथ।"
```

### डिजिटल फोरेंसिक जाँच

```
आप: "suspect.png और suspect.wav का गुप्त संचार के लिए विश्लेषण करें"

एजेंट: -> img_detect: कोई LSB स्टेगो नहीं मिला, लेकिन जुड़ा हुआ डेटा मिला
          (PNG IEND मार्कर के बाद 12,847 बाइट)
       -> file_identify: जुड़ा हुआ डेटा ZIP सिग्नेचर (PK) से शुरू होता है
       -> img_metadata: PNG tEXt चंक में असामान्य कुंजी "comm" है
          Base64 मान के साथ
       -> audio_detect: WAV फ़ाइल में LSB काई-स्क्वेयर विसंगति (p=0.003)
       -> audio_silence: सक्रिय LSB वाले 3 मौन खंड पाए गए
       -> audio_lsb_extract: 2,048 बाइट छिपा हुआ डेटा निकाला गया

       "दोनों फ़ाइलों में छिपा हुआ डेटा है:
        1. suspect.png में PNG EOF मार्कर के बाद एक ZIP आर्काइव
           जुड़ा है, साथ ही मेटाडेटा में Base64-एनकोडेड संदेश
        2. suspect.wav में LSB स्टेगनोग्राफी है, विशेष रूप से
           मौन खंडों में — जानबूझकर छिपाने का एक मज़बूत संकेत।
        ZIP निकालने और WAV पेलोड विश्लेषण करने की अनुशंसा।"
```

### ज़ीरो-विड्थ टेक्स्ट पहचान

```
आप: "यह ईमेल सामान्य दिखता है लेकिन इसमें छिपा हुआ टेक्स्ट हो सकता है"
```

## CLI उपयोग

```bash
# सहायता दिखाएँ
npx -y steganography-mcp --help

# विवरण सहित सभी 128 टूलों की सूची
npx -y steganography-mcp --list

# इमेज में स्टेगनोग्राफी का पता लगाएँ
npx -y steganography-mcp --tool img_detect '{"file_path":"challenge.png"}'

# LSB से छिपा हुआ संदेश निकालें
npx -y steganography-mcp --tool img_lsb_extract '{"file_path":"stego.png"}'

# काई-स्क्वेयर स्टेगैनालिसिस
npx -y steganography-mcp --tool img_chi_square '{"file_path":"suspect.png"}'

# RS विश्लेषण (Fridrich-Goljan-Du विधि)
npx -y steganography-mcp --tool img_rs_analysis '{"file_path":"suspect.png"}'

# JPEG डबल कम्प्रेशन पहचान
npx -y steganography-mcp --tool jpeg_double_compression '{"file_path":"photo.jpg"}'

# गहन EXIF विश्लेषण
npx -y steganography-mcp --tool jpeg_exif_deep '{"file_path":"photo.jpg"}'

# ऑडियो स्टेगनोग्राफी पहचान
npx -y steganography-mcp --tool audio_detect '{"file_path":"message.wav"}'

# ज़ीरो-विड्थ कैरेक्टर एनकोडिंग का पता लगाएँ
npx -y steganography-mcp --tool text_zwc_detect '{"text":"suspicious text here"}'

# ज़ीरो-विड्थ कैरेक्टर के साथ छिपा हुआ संदेश एम्बेड करें
npx -y steganography-mcp --tool text_zwc_embed '{"text":"cover text","message":"secret"}'

# फ़ाइल प्रकार की पहचान करें और पॉलीग्लॉट का पता लगाएँ
npx -y steganography-mcp --tool file_polyglot '{"file_path":"suspicious.pdf"}'

# एम्बेडेड फ़ाइलों के लिए स्कैन करें (binwalk-शैली)
npx -y steganography-mcp --tool file_embedded '{"file_path":"mystery.bin"}'

# एन्ट्रॉपी विज़ुअलाइज़ेशन
npx -y steganography-mcp --tool file_entropy_visual '{"file_path":"data.bin"}'

# स्वचालित एनकोडिंग पहचान
npx -y steganography-mcp --tool crypto_detect '{"input":"aGVsbG8gd29ybGQ="}'

# XOR ब्रूट-फ़ोर्स
npx -y steganography-mcp --tool crypto_xor '{"input":"4f5243484e"}'

# सिफ़र पैटर्न का पता लगाएँ
npx -y steganography-mcp --tool crypto_patterns '{"input":"Gur dhvpx oebja sbk"}'

# Bun का उपयोग करें (तेज़ स्टार्टअप)
bunx steganography-mcp --tool img_detect '{"file_path":"image.png"}'
```

## उपयोग के मामले

### CTF चैलेंज
कैप्चर-द-फ़्लैग प्रतियोगिताओं में स्टेगनोग्राफी चैलेंज हल करें। AI एजेंट व्यवस्थित रूप से सभी पहचान तकनीकें लागू कर सकता है &mdash; LSB विश्लेषण, मेटाडेटा निरीक्षण, जुड़ा हुआ डेटा, एनकोडिंग पहचान और सिफ़र शनाख्त &mdash; इमेज, ऑडियो फ़ाइलों, दस्तावेज़ों और टेक्स्ट में छिपे हुए फ़्लैग खोजने के लिए।

### डिजिटल फोरेंसिक्स
फोरेंसिक जाँच में गुप्त संचार चैनलों का पता लगाएँ। सांख्यिकीय स्टेगैनालिसिस (काई-स्क्वेयर, RS विश्लेषण) का उपयोग करके संदिग्ध फ़ाइलों में छिपे हुए डेटा का विश्लेषण करें, EOF मार्कर के बाद जुड़े हुए डेटा की जाँच करें, एम्बेडेड फ़ाइलों को स्कैन करें और स्टेगनोग्राफी टूल सिग्नेचर की पहचान करें।

### सुरक्षा अनुसंधान
स्टेगनोग्राफी टूल और तकनीकों का विश्लेषण करें। मूल और स्टेगो इमेज की पिक्सेल-दर-पिक्सेल तुलना करें, JPEG स्टेगो में DCT गुणांक वितरण का अध्ययन करें, एम्बेडिंग से एन्ट्रॉपी परिवर्तन मापें और एनकोडिंग योजनाओं को रिवर्स-इंजीनियर करें।

### शिक्षा
जानें कि स्टेगनोग्राफी तकनीकें कैसे काम करती हैं। LSB संदेश एम्बेड और एक्सट्रैक्ट करें, ज़ीरो-विड्थ कैरेक्टर के साथ टेक्स्ट एनकोड करें, बिट प्लेन और एन्ट्रॉपी मैप विज़ुअलाइज़ करें, हेक्स डंप के साथ फ़ाइल संरचनाओं का विश्लेषण करें और फ़्रीक्वेंसी विश्लेषण के साथ सिफ़र पैटर्न का अध्ययन करें।

### घटना प्रतिक्रिया
घटना प्रतिक्रिया के दौरान, छिपे हुए डेटा एक्सफ़िलट्रेशन चैनलों के लिए दस्तावेज़ों और इमेज की जाँच करें। छिपे हुए JavaScript और एम्बेडेड फ़ाइलों के लिए PDF स्कैन करें, ईमेल में ज़ीरो-विड्थ कैरेक्टर एनकोडिंग का पता लगाएँ, पॉलीग्लॉट फ़ाइलों की पहचान करें और संदिग्ध एनकोडिंग का विश्लेषण करें।

```
src/
  index.ts                    # CLI एंट्रीपॉइंट (--help, --list, --tool, stdio सर्वर)
  protocol/
    mcp-server.ts             # MCP सर्वर सेटअप (stdio ट्रांसपोर्ट)
    tools.ts                  # टूल रजिस्ट्री — सभी 128 टूल यहाँ एकत्रित
  types/
    index.ts                  # साझा प्रकार (ToolDef, ToolContext, ToolResult)
  utils/
    binary.ts                 # बाइनरी फ़ाइल रीडिंग, हेक्स डंप, फ़ॉर्मेट पहचान
    stats.ts                  # शैनन एन्ट्रॉपी, काई-स्क्वेयर, बाइट फ़्रीक्वेंसी
    cache.ts                  # TTL कैश
    png-parser.ts             # शुद्ध TS PNG पार्सर (IHDR, चंक, पिक्सेल डेटा)
    jpeg-parser.ts            # शुद्ध TS JPEG पार्सर (मार्कर, EXIF, क्वांटाइज़ेशन)
    wav-parser.ts             # शुद्ध TS WAV पार्सर (RIFF चंक, PCM सैंपल)
    bmp-parser.ts             # शुद्ध TS BMP पार्सर (हेडर, पिक्सेल डेटा)
    avi-parser.ts             # शुद्ध TS AVI पार्सर (RIFF/AVI फ्रेम)
    gif-parser.ts             # शुद्ध TS GIF पार्सर (पैलेट, LZW, एक्सटेंशन)
    pcap-parser.ts            # शुद्ध TS PCAP पार्सर (पैकेट, हेडर)
    mp3-parser.ts             # शुद्ध TS MP3 पार्सर (MPEG फ्रेम, ID3)
    zip-parser.ts             # शुद्ध TS ZIP पार्सर (एंट्री, सेंट्रल डायरेक्टरी)
  image/                      # इमेज स्टेगैनालिसिस टूल (14)
  jpeg/                       # JPEG विश्लेषण टूल (7)
  audio/                      # ऑडियो स्टेगैनालिसिस टूल (7)
  text/                       # टेक्स्ट और यूनिकोड टूल (10)
  file/                       # फ़ाइल फोरेंसिक्स टूल (10)
  document/                   # दस्तावेज़ विश्लेषण टूल (5)
  crypto/                     # एनकोडिंग और क्रिप्टो टूल (7)
  jpegadv/                    # उन्नत JPEG विश्लेषण टूल (7)
  video/                      # वीडियो स्टेगनोग्राफी टूल (8)
  gif/                        # GIF स्टेगनोग्राफी टूल (8)
  network/                    # नेटवर्क स्टेगनोग्राफी टूल (8)
  mp3/                        # MP3 स्टेगनोग्राफी टूल (7)
  spread/                     # स्प्रेड स्पेक्ट्रम टूल (5)
  bpcs/                       # BPCS विश्लेषण टूल (5)
  archive/                    # आर्काइव स्टेगनोग्राफी टूल (7)
  create/                     # निर्माण और एम्बेडिंग टूल (7)
  qrcode/                     # QR कोड स्टेगनोग्राफी टूल (6)
  data/
    encoding-patterns.ts      # एनकोडिंग regex पैटर्न + डिकोडर
```

**डिज़ाइन निर्णय:**

- **4 डिपेंडेंसी, और कुछ नहीं** &mdash; MCP प्रोटोकॉल के लिए `@modelcontextprotocol/sdk`, इनपुट सत्यापन के लिए `zod`, PNG पिक्सेल एक्सेस के लिए `pngjs`, JPEG डिकोडिंग के लिए `jpeg-js`। कोई फूला हुआ डिपेंडेंसी ट्री नहीं। कोई नेटिव मॉड्यूल नहीं। कोई C बाइंडिंग नहीं। कोई Python नहीं। कोई Java नहीं।
- **100% ऑफ़लाइन** &mdash; हर टूल पूरी तरह से स्थानीय रूप से चलता है। कोई HTTP अनुरोध नहीं। कोई API कॉल नहीं। कोई टेलीमेट्री नहीं। कोई क्लाउड डिपेंडेंसी नहीं। आपकी फ़ाइलें कभी आपकी मशीन से बाहर नहीं जातीं।
- **शुद्ध TypeScript सांख्यिकीय विश्लेषण** &mdash; काई-स्क्वेयर टेस्ट, RS विश्लेषण (Fridrich-Goljan-Du), सैंपल पेयर एनालिसिस, शैनन एन्ट्रॉपी, Index of Coincidence और फ़्रीक्वेंसी विश्लेषण सभी शुद्ध TypeScript में कार्यान्वित हैं। कोई बाहरी गणित लाइब्रेरी नहीं।
- **कस्टम फ़ॉर्मेट पार्सर** &mdash; PNG चंक, JPEG मार्कर/EXIF/क्वांटाइज़ेशन टेबल, WAV RIFF चंक और BMP हेडर `utils/` पार्सर का उपयोग करके शून्य बाहरी डिपेंडेंसी के साथ पार्स किए जाते हैं। यह गहन फ़ॉर्मेट-विशिष्ट विश्लेषण की अनुमति देता है जो सामान्य-उद्देश्य लाइब्रेरी प्रदान नहीं कर सकतीं।
- **17 प्रोवाइडर, 1 सर्वर** &mdash; प्रत्येक विश्लेषण श्रेणी एक स्वतंत्र मॉड्यूल है। AI एजेंट जाँच के संदर्भ के आधार पर कौन से टूल उपयोग करने हैं यह चुनता है।
- **स्वच्छ ToolDef पैटर्न** &mdash; हर टूल एक ही `{ name, description, schema, execute }` पैटर्न का पालन करता है। नया टूल जोड़ना उपयुक्त मॉड्यूल में एक ही ऑब्जेक्ट है।
- **हर फ़ील्ड पर Zod सत्यापन** &mdash; हर स्कीमा फ़ील्ड में AI एजेंट संदर्भ के लिए `.describe()` है। अमान्य इनपुट स्पष्ट त्रुटि संदेशों के साथ निष्पादन से पहले पकड़े जाते हैं।
